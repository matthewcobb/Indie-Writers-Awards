In app Data

If you're hosting the frontend assets on our side you can leverage Handlebars syntax to inject your app and logged user data in your page. Simply use the curly brackets signature with the name of the property you want to display and we'll do the rest. Examples:
\{{appId}} -> the app's appId \{{appRecipe.ga}} -> Google Analytics ID \{{user.displayName}} -> the user's name
